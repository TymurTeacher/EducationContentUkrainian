# Діяльність 3 - Яке правильне число?

## Крок 1
Подивіться на кожен з екранів телевізора, обчисліть суми за допомогою коду.
Щоб використовувати математичне значення чисел у команді, лапки **не повинні** використовуватися. Наприклад:
```python
player.say(1+1) 
```

Коли цей код запускається, він дає результат **2**. Визначте правильні відповіді сум на екранах телевізора та виберіть відповідні кнопки.
